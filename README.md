# Пара ближайших точек
Необходимо написать функцию, ищущую в массиве пару точек, ближе всего расположенных к друг другу.
Функция принимает на вход массив пар координат, а на выходе должна вернуть кортеж из расстояния,
координат первой точки и координат второй точки.
Результирующие точки могут быть возвращены в любом порядке. Если пар точек, расстояние между которыми
наименьшее, несколько, можно вернуть любую из таких пар.
